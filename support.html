Frequently Asked Questions

How do I enable the keyboard?

Answer:

  1. Open iOS Settings > General > Keyboard > Keyboards
  2. Tap "Add New Keyboard"
  3. Select "UnderText"
  4. Enable "Allow Full Access" for translation features

Why does the app need Full Access?

Answer: Full Access is required for:

  • Real-time translation as you type
  • Translation button functionality
  • Connecting to AI translation services

Privacy Note: We do NOT collect, store, or transmit your keystrokes. See our Privacy Policy for details.

How does the 3-day free trial work?

Answer:

  • Trial starts automatically on first app launch
  • No payment information required
  • Full access to all premium features for 3 days
  • After trial expires, translation features are locked
  • Subscribe anytime to unlock features again

How do I subscribe to Premium?

Answer:

  1. Open the UnderText main app
  2. Tap "Upgrade to Premium" in Settings
  3. Choose the yearly subscription ($39/year)
  4. Confirm purchase with Apple App Store

How do I cancel my subscription?

Answer:

  1. Open iOS Settings > [Your Name] > Subscriptions
  2. Select "UnderText"
  3. Tap "Cancel Subscription"

Note: You'll continue to have access until the end of your billing period.

How do I set up Microsoft Translator API?

Answer:

  1. Go to Azure Portal
  2. Create a free Translator resource
  3. Important: Choose "Global" for the region
  4. Go to "Keys and Endpoint" section
  5. Copy "Key 1"
  6. Paste it in UnderText Settings > AI & Translation > Microsoft Translator

Free tier: 2 million characters per month

How do I set up OpenAI/Gemini/DeepSeek?

Answer:

  • OpenAI: Get API key at platform.openai.com/api-keys
  • Google Gemini: Get API key at aistudio.google.com/app/apikey
  • DeepSeek: Get API key at platform.deepseek.com/api_keys

Then paste the API key in UnderText Settings > AI & Translation > Choose your provider.

Which translation service should I use?

Answer:

  • Apple Translation: Most private (on-device), no API key needed, free, offline capable
  • Microsoft Translator: 2M free characters/month, good for most users
  • OpenAI GPT: Best quality, context-aware, costs per usage
  • Google Gemini: Fast, large context, generous free tier
  • DeepSeek: Most affordable AI option

Translation is not working. What should I check?

Answer: Troubleshooting checklist:

  1. Check if you have an active subscription or trial
  2. Make sure "Allow Full Access" is enabled
  3. For Apple Translation: Download language packs in iOS Translate app
  4. For third-party services: Verify API key is entered correctly
  5. Check your internet connection (required for cloud services)
  6. Try the "Test Translation" button in Settings

Can I get a refund?

Answer: All purchases are processed through Apple App Store. To request a refund:

  1. Go to reportaproblem.apple.com
  2. Sign in with your Apple ID
  3. Find UnderText subscription
  4. Select "Request a refund"

Refunds are subject to Apple's refund policy.
