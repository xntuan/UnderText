What is Code Switching?

Code switching lets you mix languages in one message - perfect for bilingual conversations or when you can't remember a word in one language.

How to Change Languages:

  1. Open the UnderText main app
  2. Go to Settings
  3. Find Language Settings
  4. Set your Source Language (the language you type)
  5. Set your Target Language (the language you're learning)
  6. The keyboard will automatically detect which language you're typing
